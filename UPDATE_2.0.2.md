## Как это работает

### Отмена записи

**Пользователь:**
1. Нажимает "❌ Отменить запись"
2. Видит информацию о записи
3. Подтверждает отмену
4. Запись отменена

**Если нет активной записи:**
```
Бот: ❌ У вас нет активных записей.
```

**Если есть активная запись:**
```
Бот: ⚠️ Подтверждение отмены

     Вы действительно хотите отменить запись?
     
     📅 15.04.2026 в 14:00
     
     [Кнопка: ✅ Да, отменить]
     [Кнопка: ❌ Нет, оставить]
```

### Удаление профиля

**Пользователь:**
1. Нажимает "👤 Мой профиль"
2. Видит кнопку "🗑 Удалить профиль"
3. Нажимает → видит предупреждение
4. Подтверждает → профиль удалён

**Процесс удаления:**
```
Пользователь: [Нажимает "🗑 Удалить профиль"]

Бот: ⚠️ Удаление профиля
     
     Вы уверены, что хотите удалить свой профиль?
     
     ⚠️ Это действие нельзя отменить!
     Будут удалены:
     • Ваше имя и телефон
     • Дополнительная информация
     • История кодов записи
     
     Активные записи НЕ будут отменены.
     
     [Кнопка: ✅ Да, удалить]
     [Кнопка: ❌ Отмена]

Пользователь: [Нажимает "✅ Да, удалить"]

Бот: ✅ Профиль удалён
     
     Ваш профиль успешно удалён.
     
     Вы можете создать новый профиль в любое время через /edit_profile
```

---

## Что удаляется при удалении профиля

### ✅ Удаляется:
- Имя пользователя
- Номер телефона
- Дополнительная информация
- Вся история кодов записи
- Запись в таблице `user_profiles`
- Все записи в таблице `booking_codes`

### ❌ НЕ удаляется:
- Активные записи (appointments)
- История отзывов
- Статус подписки
