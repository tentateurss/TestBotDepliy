## Быстрый старт

1. Получите токен бота от @BotFather в Telegram
2. Откройте `src/main/resources/application.properties`
3. Замените `YOUR_BOT_TOKEN_HERE` на ваш токен
4. Замените `YOUR_BOT_USERNAME_HERE` на имя вашего бота
5. Замените `YOUR_ADMIN_ID` на ваш Telegram ID
6. Запустите бота:
   - Windows: двойной клик на `start.bat`
   - Linux/Mac: `chmod +x start.sh && ./start.sh`
