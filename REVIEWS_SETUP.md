# Инструкция по настройке канала с отзывами

## Что изменилось

✅ **Система отзывов переделана:**
- Клиент нажимает кнопку "⭐️ Отзывы" в главном меню
- Видит ссылку на канал с отзывами, где может посмотреть отзывы других клиентов
- После процедуры бот просит оставить отзыв
- Клиент пишет отзыв прямо в боте
- Бот автоматически пересылает отзыв в канал с отзывами

✅ **Добавлена кнопка "📋 Мои записи":**
- Клиент может посмотреть историю своих записей
- Если записей нет, показывается сообщение "У вас пока нет записей"

✅ **Убран функционал быстрой повторной записи**

## Настройка канала с отзывами

### Шаг 1: Создайте канал для отзывов

1. Откройте Telegram
2. Создайте новый канал (например, "Отзывы о мастере")
3. Сделайте канал публичным и задайте ссылку (например, @your_reviews_channel)

### Шаг 2: Добавьте бота в канал

1. Откройте настройки канала
2. Перейдите в "Администраторы"
3. Нажмите "Добавить администратора"
4. Найдите вашего бота (@Valakas_bot_bot)
5. Дайте боту права на публикацию сообщений

### Шаг 3: Получите ID канала

1. Перешлите любое сообщение из канала боту @userinfobot
2. Бот ответит с информацией о канале
3. Скопируйте ID канала (например: -1001234567890)

### Шаг 4: Настройте переменные окружения

Откройте файл `.env` и обновите настройки канала отзывов:

```properties
# Reviews Channel Configuration
BOT_REVIEWS_CHANNEL_ID=-1001234567890
BOT_REVIEWS_CHANNEL_LINK=https://t.me/your_reviews_channel
```

**Замените:**
- `-1001234567890` на реальный ID вашего канала
- `https://t.me/your_reviews_channel` на реальную ссылку на ваш канал

### Пример правильной настройки:

```properties
BOT_REVIEWS_CHANNEL_ID=-1002345678901
BOT_REVIEWS_CHANNEL_LINK=https://t.me/lashes_reviews
```

### Альтернативный способ (application.properties):

Если вы не используете .env файл, можете указать значения напрямую в `src/main/resources/application.properties`:

```properties
bot.reviews.channel.id=-1002345678901
bot.reviews.channel.link=https://t.me/lashes_reviews
```

## Как это работает

### Для клиента:

1. **Просмотр отзывов:**
   - Клиент нажимает "⭐️ Отзывы" в главном меню
   - Видит кнопку "📖 Посмотреть отзывы"
   - Нажимает и переходит в канал с отзывами

2. **Оставить отзыв:**
   - После процедуры бот автоматически просит оставить отзыв
   - Клиент выбирает оценку (1-5 звёзд)
   - Пишет текст отзыва
   - Отзыв публикуется в канале

3. **История записей:**
   - Клиент нажимает "📋 Мои записи"
   - Видит список всех своих записей (активных и отменённых)
   - Если записей нет, видит сообщение "У вас пока нет записей"

### Для администратора:

- Все отзывы автоматически публикуются в канале
- Можно удалять неподходящие отзывы вручную
- Отзывы сохраняются в базе данных

## Структура главного меню

Теперь главное меню выглядит так:

```
┌─────────────────────┐
│   📝 Записаться     │
├──────────┬──────────┤
│ 💰 Прайс │⭐️ Отзывы│
├──────────┼──────────┤
│🖼 Портфолио│📋 Мои записи│
└──────────┴──────────┘
```

Для админа добавляется ещё одна строка:
```
┌─────────────────────┐
│ ⚙️ Админ-панель     │
└─────────────────────┘
```

## Проверка работы

После настройки проверьте:

1. ✅ Кнопка "⭐️ Отзывы" появилась в главном меню
2. ✅ При нажатии показывается ссылка на канал
3. ✅ Бот является администратором канала с отзывами
4. ✅ После создания тестовой записи бот просит оставить отзыв
5. ✅ Отзыв публикуется в канале
6. ✅ Кнопка "📋 Мои записи" показывает историю

## Формат отзыва в канале

Отзывы публикуются в таком формате:

```
⭐️ Новый отзыв

👤 Имя клиента
⭐️⭐️⭐️⭐️⭐️

💬 Текст отзыва клиента
```

## Важно

- Бот должен быть администратором канала с правами на публикацию
- ID канала всегда начинается с `-100`
- Ссылка на канал должна быть в формате `https://t.me/channel_name`
- Если настройки не указаны, кнопка "Отзывы" будет показывать сообщение "Раздел отзывов временно недоступен"
