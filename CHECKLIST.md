## 🐛 Решение проблем

### Бот не запускается

```
❌ Проблема: "Failed to initialize database"
✅ Решение: Проверьте путь database.path в application.properties

❌ Проблема: "Invalid bot token"
✅ Решение: Проверьте bot.token в application.properties

❌ Проблема: "getXXX() is undefined"
✅ Решение: Build → Rebuild Project (Lombok не сработал)
```

### Отзывы не публикуются

```
❌ Проблема: Отзывы не появляются в канале
✅ Решение:
   1. Проверьте bot.reviews.channel.id
   2. Убедитесь, что бот - администратор канала
   3. Проверьте права бота на публикацию
   4. Проверьте логи на ошибки
```

### WebApp не открывается

```
❌ Проблема: "Failed to load WebApp"
✅ Решение:
   1. Проверьте, что URL начинается с https://
   2. Проверьте доступность URL в браузере
   3. Откройте консоль браузера (F12) для ошибок
   4. Проверьте настройки CORS
```

### Напоминания не приходят

```
❌ Проблема: Клиент не получает напоминания
✅ Решение:
   1. Проверьте логи - должны быть записи о планировании
   2. Убедитесь, что Quartz Scheduler запущен
   3. Проверьте, что запись создана корректно
   4. Проверьте время записи (должно быть в будущем)
```
