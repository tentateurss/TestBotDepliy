# Инструкция для пользователей бота

## Первый запуск

### Шаг 1: Создайте профиль
Перед первой записью необходимо заполнить профиль:

1. Отправьте команду `/edit_profile`
2. Введите ваше имя
3. Введите номер телефона
4. (Опционально) Укажите дополнительную информацию или отправьте "-" чтобы пропустить

✅ Профиль сохранён! Теперь эти данные будут использоваться автоматически при каждой записи.

## Как записаться

### Шаг 2: Процесс записи
1. Нажмите кнопку **"📝 Записаться"**
2. Выберите **дату** из календаря
3. Выберите **время** из доступных слотов
4. Выберите **процедуру** из списка
5. Оплатите предоплату 50% по указанным реквизитам
6. Отправьте скриншот оплаты в чат
7. Дождитесь подтверждения от администратора

✅ После подтверждения вы получите **уникальный код записи** (например: X7G3H9)

## Полезные команды

- `/profile` - посмотреть свой профиль и историю записей
- `/edit_profile` - изменить данные профиля (имя, телефон, доп. информацию)
- `/start` - вернуться в главное меню

## Преимущества системы профилей

✨ **Быстрая запись** - не нужно каждый раз вводить имя и телефон  
✨ **История записей** - все ваши коды записи сохраняются в профиле  
✨ **Удобное управление** - можно обновить данные в любой момент  
✨ **Безопасность** - уникальный код для каждой записи  

## Часто задаваемые вопросы

**Q: Что делать, если я изменил номер телефона?**  
A: Просто отправьте `/edit_profile` и обновите данные.

**Q: Можно ли записаться без заполнения профиля?**  
A: Нет, профиль необходим для записи. Это занимает всего минуту!

**Q: Где хранятся мои данные?**  
A: Все данные хранятся локально в базе данных бота и не передаются третьим лицам.

**Q: Зачем нужен код записи?**  
A: Код записи - это ваш уникальный идентификатор для конкретной записи. Вы можете использовать его при общении с администратором.

**Q: Могу ли я иметь несколько активных записей?**  
A: Нет, можно иметь только одну активную запись. Для новой записи нужно сначала отменить текущую.

## Пример использования

```
1. Первый раз:
   /edit_profile → Ввод данных → Профиль создан ✅

2. Каждая запись:
   📝 Записаться → Дата → Время → Процедура → Оплата → Код ✅

3. Просмотр истории:
   /profile → Видите все свои коды записей
```

💡 **Совет:** Сохраните код записи - он может понадобиться при общении с администратором!
