## ⚠️ Важные замечания

1. **Канал отзывов не настроен**
   - В `.env` указаны placeholder значения для канала отзывов
   - Необходимо создать канал и обновить настройки:
     ```
     BOT_REVIEWS_CHANNEL_ID=-1002XXXXXXXXX
     BOT_REVIEWS_CHANNEL_LINK=https://t.me/your_reviews_channel
     ```

2. **Lombok должен быть включен**
   - В IntelliJ IDEA: Settings → Annotation Processors → Enable
   - Иначе будут ошибки компиляции

3. **База данных**
   - При первом запуске Flyway создаст все таблицы
   - Если БД уже существует, Flyway применит только новые миграции
   - Версия БД логируется при запуске

4. **Переменные окружения**
   - Файл `.env` НЕ коммитится в Git (добавлен в .gitignore)
   - При деплое на сервер настройте переменные окружения там
   - Или используйте значения по умолчанию в `application.properties`
