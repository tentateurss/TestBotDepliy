# Обновление 2.0.1 - Добавлена кнопка "Мой профиль"

## Дата: 12 апреля 2026, 15:31

### Что изменилось

Добавлена кнопка **"👤 Мой профиль"** в главное меню бота для удобного доступа к профилю пользователя.

### Изменения в коде

#### 1. Constants.java
Добавлена новая константа:
```java
public static final String BTN_MY_PROFILE = "👤 Мой профиль";
```

#### 2. ReplyKeyboardFactory.java
Добавлена кнопка в главное меню:
```java
KeyboardRow row4 = new KeyboardRow();
row4.add(Constants.BTN_MY_PROFILE);
keyboard.add(row4);
```

#### 3. MessageRouter.java
Добавлена обработка нажатия кнопки:
```java
case Constants.BTN_MY_PROFILE:
    profileHandler.handleViewProfile(msg, bot);
    break;
```

### Как это работает

**Для пользователя:**

1. Открывает бота
2. Видит кнопку **"👤 Мой профиль"** в главном меню
3. Нажимает на кнопку
4. Видит свой профиль с данными и историей записей
5. Может нажать **"✏️ Редактировать профиль"** для изменения данных

**Структура меню:**

```
┌─────────────────────────────────┐
│   📱 Открыть приложение          │
├─────────────────┬───────────────┤
│ 📝 Записаться   │ 💰 Прайс      │
├─────────────────┼───────────────┤
│ 🖼 Портфолио    │ ⭐️ Отзывы     │
├─────────────────────────────────┤
│      👤 Мой профиль              │
└─────────────────────────────────┘
```

### Преимущества

✅ **Удобный доступ** - не нужно вводить команду `/profile`  
✅ **Интуитивно понятно** - кнопка всегда на виду  
✅ **Быстрое редактирование** - один клик до редактирования профиля  
✅ **История записей** - все коды в одном месте  

### Что может делать пользователь

**При нажатии на "👤 Мой профиль":**

1. **Если профиль заполнен:**
   - Видит имя, телефон, дополнительную информацию
   - Видит последние 5 кодов записи
   - Может нажать "✏️ Редактировать профиль"

2. **Если профиль не заполнен:**
   - Видит сообщение о необходимости создать профиль
   - Получает инструкцию использовать `/edit_profile`

**При нажатии на "✏️ Редактировать профиль":**
- Может изменить имя
- Может изменить телефон
- Может изменить дополнительную информацию
