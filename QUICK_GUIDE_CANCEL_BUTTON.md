# 🚀 Быстрая инструкция: Кнопка "Отменить запись"

## Что изменилось?

Теперь кнопка **"❌ Отменить запись"** появляется в главном меню **автоматически**, когда у клиента есть активная запись.

---

## 📱 Как это работает для клиента?

### 1. До записи
```
┌─────────────────────────────┐
│ 📝 Записаться  │ 💰 Прайс   │
│ 🖼 Портфолио   │ ⭐️ Отзывы │
└─────────────────────────────┘
```

### 2. После записи
```
┌─────────────────────────────┐
│ 📝 Записаться  │ 💰 Прайс   │
│ 🖼 Портфолио   │ ⭐️ Отзывы │
│ ❌ Отменить запись          │ ← Появилась!
└─────────────────────────────┘
```

### 3. Процесс отмены
1. Клиент нажимает **"Отменить запись"**
2. Подтверждает отмену
3. Запрос отправляется администратору
4. Администратор принимает решение:
   - ✅ **Подтверждает** → запись отменена, кнопка исчезает
   - ❌ **Отклоняет** → запись остается, кнопка остается

---

## ⚙️ Для администратора

### Уведомление об отмене
Когда клиент запрашивает отмену, вы получите сообщение:

```
⚠️ ЗАПРОС НА ОТМЕНУ ЗАПИСИ

👤 Клиент: Анна
📱 Телефон: +7 999 123 45 67
📅 Дата: 17.04.2026
⏰ Время: 14:00
💎 Процедура: Классика

[✅ Подтвердить отмену]  [❌ Отклонить]
```

### Действия
- **Подтвердить** - запись отменяется, слот освобождается
- **Отклонить** - запись остается активной
