# 🚀 ДЕПЛОЙ БОТА

## Вариант 1: Railway.app (Рекомендуется)

```bash
# Установи Railway CLI
npm i -g @railway/cli

# Залогинься
railway login

# Инициализируй проект
railway init

# Задеплой
railway up

# Проверь логи
railway logs
```

## Вариант 2: Render.com

1. Зарегистрируйся на https://render.com
2. Создай новый Web Service
3. Подключи Git репозиторий
4. Выбери Docker
5. Deploy!
