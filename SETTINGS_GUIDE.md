# Руководство по настройкам бота

## 📋 Обзор

В админ-панели добавлен раздел **"⚙️ Настройки бота"**, который позволяет мастеру полностью настроить бота под свои нужды без изменения кода.

## 🎯 Доступные настройки

### 1. ✏️ Приветственное сообщение

**Как использовать:**
1. Откройте админ-панель: `/admin`
2. Нажмите "⚙️ Настройки бота"
3. Нажмите "✏️ Приветственное сообщение"
4. Отправьте новое приветственное сообщение

**Возможности:**
- Добавьте свой адрес и время работы
- Используйте Markdown для форматирования:
  - `*жирный текст*`
  - `_курсив_`
  - `[ссылка](https://example.com)`
- Используйте `%s` для автоматической подстановки имени пользователя

**Пример:**
```
🌸 *Добро пожаловать, %s!*

Я помогу вам записаться на процедуру наращивания ресниц.

📍 *Адрес:* ул. Ленина, д. 10, офис 5
🕐 *Время работы:* Пн-Пт 10:00-20:00, Сб-Вс 11:00-18:00
📱 *Телефон:* +7 (999) 123-45-67

Выберите действие в меню:
```

**Где используется:**
- При команде `/start`
- При первом запуске бота пользователем

---

### 2. 🔗 Управление ссылками

**Как открыть:**
1. Откройте админ-панель: `/admin`
2. Нажмите "⚙️ Настройки бота"
3. Нажмите "🔗 Управление ссылками"

#### 2.1. Обязательная подписка на канал

**Переключатель:** "✅ Подписка: ВКЛ" / "❌ Подписка: ВЫКЛ"

**Когда включено:**
- Пользователи должны подписаться на ваш канал перед записью
- Бот проверяет подписку автоматически
- Кнопка "📢 Изменить канал подписки" становится доступной

**Когда выключено:**
- Пользователи могут записываться без подписки
- Кнопка изменения канала скрыта

**Как изменить ссылку на канал:**
1. Включите обязательную подписку
2. Нажмите "📢 Изменить канал подписки"
3. Отправьте ссылку на канал (формат: `https://t.me/your_channel`)

**Пример:** `https://t.me/lashes_beauty`

---

#### 2.2. Функция отзывов

**Переключатель:** "✅ Отзывы: ВКЛ" / "❌ Отзывы: ВЫКЛ"

**Когда включено:**
- Кнопка "⭐️ Отзывы" отображается в главном меню
- Пользователи могут оставлять отзывы после процедуры
- Отзывы публикуются в вашем канале
- Кнопка "⭐️ Изменить канал отзывов" становится доступной

**Когда выключено:**
- Кнопка "⭐️ Отзывы" полностью скрыта из главного меню
- Функция отзывов отключена
- Кнопка изменения канала скрыта

**Как изменить ссылку на канал отзывов:**
1. Включите функцию отзывов
2. Нажмите "⭐️ Изменить канал отзывов"
3. Отправьте ссылку на канал (формат: `https://t.me/your_reviews`)

**Пример:** `https://t.me/lashes_reviews`

## 🔧 Технические детали

### Хранение настроек

Все настройки хранятся в базе данных в таблице `settings`:

| Ключ | Тип | По умолчанию | Описание |
|------|-----|--------------|----------|
| `welcome_message` | String | Стандартное приветствие | Приветственное сообщение |
| `subscription_required` | Boolean | `false` | Обязательная подписка |
| `subscription_channel_link` | String | - | Ссылка на канал подписки |
| `reviews_enabled` | Boolean | `true` | Функция отзывов включена |
| `reviews_channel_link` | String | - | Ссылка на канал отзывов |

### Где используются настройки

**Приветственное сообщение:**
- `CommonHandler.java` - метод `handleStart()`

**Обязательная подписка:**
- `BookingHandler.java` - метод `handleBook()` (проверка перед записью)
- `InlineKeyboardFactory.java` - метод `getSubscriptionCheckKeyboard()` (ссылка на канал)

**Функция отзывов:**
- `ReplyKeyboardFactory.java` - метод `getMainMenuWithWebApp()` (показ/скрытие кнопки)

## 📱 Примеры использования

### Сценарий 1: Настройка для нового мастера

1. **Установите приветственное сообщение:**
   ```
   👋 Привет, %s!
   
   Я бот для записи к мастеру Анне.
   
   📍 Адрес: ТЦ "Галерея", 2 этаж
   🕐 Работаю: Вт-Сб 10:00-19:00
   ```

2. **Включите обязательную подписку:**
   - Переключите "Подписка" в положение ВКЛ
   - Укажите ссылку на ваш канал: `https://t.me/anna_lashes`

3. **Настройте отзывы:**
   - Убедитесь, что "Отзывы" включены
   - Укажите ссылку на канал отзывов: `https://t.me/anna_reviews`

### Сценарий 2: Работа без подписки и отзывов

Если вы хотите простой бот без дополнительных функций:

1. **Выключите обязательную подписку:**
   - Переключите "Подписка" в положение ВЫКЛ

2. **Выключите отзывы:**
   - Переключите "Отзывы" в положение ВЫКЛ
   - Кнопка "⭐️ Отзывы" исчезнет из меню

### Сценарий 3: Изменение адреса

Если вы переехали в новое место:

1. Откройте "⚙️ Настройки бота"
2. Нажмите "✏️ Приветственное сообщение"
3. Отправьте обновлённое сообщение с новым адресом

## ⚠️ Важные замечания

1. **Формат ссылок:** Ссылки на каналы должны начинаться с `https://t.me/`
2. **Markdown:** Используйте правильный синтаксис Markdown, иначе сообщение может отображаться некорректно
3. **Подстановка имени:** Используйте `%s` в приветственном сообщении для автоматической подстановки имени пользователя
4. **Переключатели:** Изменения применяются мгновенно после нажатия
5. **Кнопки отзывов:** При выключении функции отзывов кнопка исчезает из меню всех пользователей

## 🐛 Решение проблем

**Проблема:** Приветственное сообщение отображается некорректно
- **Решение:** Проверьте синтаксис Markdown. Используйте `*` для жирного текста, `_` для курсива

**Проблема:** Ссылка на канал не сохраняется
- **Решение:** Убедитесь, что ссылка начинается с `https://t.me/`

**Проблема:** Кнопка "Отзывы" не исчезает после выключения
- **Решение:** Пользователь должен перезапустить бота командой `/start`

**Проблема:** Подписка не проверяется
- **Решение:** Убедитесь, что бот добавлен в канал как администратор
