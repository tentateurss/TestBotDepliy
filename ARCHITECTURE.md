## Поток работы клиента

```
1. Клиент открывает бота
        │
        ▼
2. Выбирает "📱 Открыть приложение"
        │
        ▼
3. Открывается WebApp с главным меню
        │
        ├─► 📝 Записаться
        │   └─► Выбор даты → Выбор времени → Ввод данных → Подтверждение
        │
        ├─► 💰 Прайс-лист
        │   └─► Просмотр цен
        │
        ├─► ⭐️ Отзывы
        │   └─► Просмотр отзывов → Переход в канал
        │
        ├─► 📋 Мои записи
        │   └─► История всех записей
        │
        └─► 🖼 Портфолио
            └─► Переход на Pinterest
```

## Поток работы администратора

```
1. Админ открывает бота
        │
        ▼
2. Выбирает "⚙️ Админ-панель"
        │
        ▼
3. Открывается админ WebApp
        │
        ├─► 📋 Записи клиентов
        │   └─► Таблица с фильтрами → Отмена записи
        │
        ├─► 📅 Управление расписанием
        │   └─► Добавить день → Шаблон → Редактировать
        │
        ├─► 📊 Статистика
        │   └─► Записи → Доход → Популярные слоты
        │
        ├─► ⭐️ Отзывы
        │   └─► Просмотр → Удаление
        │
        ├─► 💬 Рассылка
        │   └─► Написать сообщение → Выбрать аудиторию → Отправить
        │
        └─► ⚙️ Настройки
            └─► Слоты → Цены → Прайс
```

## Автоматические процессы

```
Запись создана
    │
    ├─► За 24 часа до записи
    │   └─► 📲 Напоминание клиенту
    │
    ├─► За 3 часа до записи
    │   └─► ⏰ Второе напоминание
    │
    ├─► Через 30 минут после записи
    │   └─► ⭐️ Запрос отзыва
    │       └─► Клиент пишет отзыв
    │           └─► Бот публикует в канал
    │
    └─► Через 2 часа после начала
        └─► 🗑 Автоудаление записи
```

## Технологии

```
Backend (Java):
├── Telegram Bot API (telegrambots 6.9.7.1)
├── SQLite (sqlite-jdbc 3.45.3.0)
├── Quartz Scheduler (2.3.2)
├── Jackson (2.14.2)
├── Lombok (1.18.30)
└── Logback (1.4.14)

Frontend (WebApp):
├── HTML5
├── CSS3 (Grid, Flexbox, Animations)
├── JavaScript (ES6+)
└── Telegram WebApp API

База данных:
└── SQLite 3
```

## Файлы конфигурации

```
application.properties
├── bot.token                    # Токен бота
├── bot.username                 # Имя бота
├── bot.admin.id                 # ID админов
├── bot.channel.id               # ID канала подписки
├── bot.channel.link             # Ссылка на канал
├── bot.reviews.channel.id       # ID канала отзывов ⚠️
├── bot.reviews.channel.link     # Ссылка на канал отзывов ⚠️
└── database.path                # Путь к БД
```
