## 📝 Что нужно настроить

### Обязательно: Настройте канал с отзывами

1. Создайте публичный канал в Telegram
2. Добавьте бота в канал как администратора
3. Получите ID канала через @userinfobot
4. Откройте `src/main/resources/application.properties`
5. Замените эти строки:

```properties
# ID канала с отзывами (получить через @userinfobot)
bot.reviews.channel.id=-1001234567890

# Ссылка на канал (формат: https://t.me/channel_name)
bot.reviews.channel.link=https://t.me/your_reviews_channel
```

**Пример:**
```properties
bot.reviews.channel.id=-1002345678901
bot.reviews.channel.link=https://t.me/lashes_reviews
```

## 🚀 Как запустить

1. Откройте проект в IntelliJ IDEA
2. Включите обработку аннотаций Lombok:
   - Settings → Build → Compiler → Annotation Processors
   - Поставьте галочку "Enable annotation processing"
3. Build → Rebuild Project
4. Run → Run 'BotApplication'

## 🆕 Обновление от 16.04.2026 - Настройки бота

### 5. Добавлен раздел "⚙️ Настройки бота" в админ-панели

Теперь мастер может полностью настроить бота под свои нужды без изменения кода!

#### 5.1. ✏️ Редактирование приветственного сообщения

**Что можно настроить:**
- Текст приветствия при команде /start
- Адрес салона
- Время работы
- Контактную информацию

**Возможности:**
- Поддержка Markdown форматирования (*жирный*, _курсив_)
- Автоматическая подстановка имени пользователя через `%s`
- Сохранение в базе данных

**Как использовать:**
1. Админ-панель → "⚙️ Настройки бота"
2. "✏️ Приветственное сообщение"
3. Отправьте новое сообщение

#### 5.2. 🔗 Управление ссылками

**Обязательная подписка на канал:**
- ✅/❌ Переключатель включения/выключения
- Когда включено: пользователи должны подписаться перед записью
- Когда выключено: запись доступна без подписки
- Редактирование ссылки на канал (только если включено)

**Функция отзывов:**
- ✅/❌ Переключатель включения/выключения
- Когда включено: кнопка "⭐️ Отзывы" отображается в меню
- Когда выключено: кнопка "⭐️ Отзывы" полностью скрыта
- Редактирование ссылки на канал отзывов (только если включено)

**Как использовать:**
1. Админ-панель → "⚙️ Настройки бота"
2. "🔗 Управление ссылками"
3. Нажмите на переключатель для включения/выключения
4. Измените ссылки при необходимости
