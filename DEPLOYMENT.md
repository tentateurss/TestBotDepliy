# 🚀 Инструкция по развертыванию бота для клиентов

## Что нужно для запуска

1. Java 17 или выше
2. Токен бота от @BotFather
3. Telegram ID администратора

## Быстрый старт

### Шаг 1: Создание бота в Telegram

1. Откройте @BotFather в Telegram
2. Отправьте команду `/newbot`
3. Придумайте имя и username для бота
4. Скопируйте токен (например: `123456789:ABCdefGHIjklMNOpqrsTUVwxyz`)

### Шаг 2: Узнайте свой Telegram ID

1. Откройте @userinfobot в Telegram
2. Отправьте любое сообщение
3. Скопируйте ваш ID (например: `123456789`)

### Шаг 3: Настройка бота

Откройте файл `src/main/resources/application.properties` и измените:

```properties
# Токен вашего бота
bot.token=ВАШ_ТОКЕН_ОТ_BOTFATHER

# Username вашего бота
bot.username=ваш_бот_username

# Ваш Telegram ID (можно указать несколько через запятую)
bot.admin.id=ВАШ_TELEGRAM_ID

# Супер-админы (разработчики) - НЕ УДАЛЯЙТЕ ЭТУ СТРОКУ!
bot.super.admins=964712947,968738208
```

### Шаг 4: Настройка каналов (опционально)

Если хотите уведомления в канал:

```properties
# ID канала (получить: добавьте бота в канал, перешлите сообщение из канала боту @userinfobot)
bot.channel.id=-1001234567890

# Ссылка на канал
bot.channel.link=https://t.me/your_channel

# Username канала
bot.channel.username=your_channel
```

### Шаг 5: Запуск бота

#### Windows:
```bash
mvnw.cmd clean package
java -jar target/BotTG2-1.0.0.jar
```

#### Linux/Mac:
```bash
./mvnw clean package
java -jar target/BotTG2-1.0.0.jar
```

## Первый запуск

1. Откройте вашего бота в Telegram
2. Отправьте `/start`
3. Нажмите "Админ-панель" для настройки

## Настройка через админ-панель

### Добавление рабочих дней

1. Админ-панель → "Добавить рабочий день"
2. Выберите дату в календаре
3. Введите слоты времени (например: `10:00, 11:00, 12:00`) или `default`

### Настройка расписания шаблоном

1. Админ-панель → "Настройка расписания шаблоном"
2. Выберите дни недели (Пн, Вт, Ср...)
3. Выберите часы работы
4. Выберите период (неделя, месяц...)

### Управление услугами

1. Админ-панель → "Управление прайсом"
2. Добавьте свои услуги и цены

## Важно!

⚠️ **НЕ УДАЛЯЙТЕ** строку `bot.super.admins` из конфигурации!
Это ID разработчиков, которые могут помочь с технической поддержкой.

## Обновления

Для обновления бота:
1. Остановите текущий процесс
2. Скачайте новую версию
3. Скопируйте файл `application.properties` из старой версии
4. Запустите новую версию

База данных сохранится автоматически.
